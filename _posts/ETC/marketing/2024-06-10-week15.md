## 13장. 통합 마케팅 커뮤니케이션 (촉진)

판매 촉진에는 네 가지가 있다.

- 광고
- 인적 판매 : 제품이나 서비스의 판매를 목적으로 기존 도는 잠재 고객에 대한 판매 프리젠테이션
- Pulblic Relation : 기업이 자사의 이미지를 제고하거나 자사에 대한 호의적인 평판을 얻거나 비호의적인 평판을 제거 또는 완화시키려는 커뮤니케이션 활동

발신자 : 커뮤니케이션

소비자들이 발신인이 의도한 대로 메시지를 수용하지 못하는 이유 (시험에 나올 것 같음)

- 선택적 주의 : 모든 자극을 감지하지 못함
- 선택적 왜곡 : 듣고 싶은 것만을 듣는 경향 - 메시지를 자의적으로 변형하여 수용
- 선택적 회상 : 장기 기억 속에 메시지의 일부만을 간직하고 검색할 때에도 선택적으로 검색함

촉진 전략의 방향

- Pull strategy : 제조업자가 광고나 소비자 판촉을 이용하여 최종 소비자에게 브랜드나 제품을 알려 스스로 적극적으로 구매하도록 하는 전략
- Push strategy : 제조업자가 판매촉진이나 인적 판매를 이용하여 중간상으로 하여금 제품을 구비하고 소비자에게 적극적으로 판매하도록 유도하는 전략

광고의 역할 (시험에 나올 것 같음)

- 정보 전달
- 구매 설득
- 제품이나 브랜드에 대한 우호적인 태도 형성
- 경제 발전에 공헌
- 변화의 촉매제

recall (회상), recogition (재인)

- ATL : TV, 라디오, 신문, 잡지 등의 4대 대중매체와 인터넷, 옥외광고, 케이블 TV 등

표적 청중의 파악

1. 청중 : 제품의 잠재적 구매자나 현재 사용자들, 개개 소비자들이거나 집단일 수도 있음
2. 표적 청중 결정에는 시장 세분화와 표적 시장 선택 등이 중요한 지침이 됨

표적 청중으로부터 원하는 반응을 결정

- 인지 : 표적 청중이 제품이나 기업을 인지하돌고 하는 노력
- 지식 : 표적 청중..
- 호감
- 선호
- 확신
- 구매

시험에 나올 것 같음

고관여 구매의사결정

- 학습 하이어라키 : 관여수준이 높고, 브랜드 간에 뚜렷한 차이가 있을 경우
- 인지 부조화 하이어라키 : 고관여 제품이지만 브랜드간 차이가 미미한 경우

저관여 구매의사결정

- 저관여 하이어라키 : 브랜드간 차이가 미미하며 저관여 제품인 경우

인지부조화 : 구매할 때 잘 샀을까? 후회할 수 있을까?

메시지의 선택

- 효과적인 메시지 : 주의를 끌고 흥미를 유발하고 욕구를 자극하며 행동을 이끌어낼 수 있어야 함

1 메시지 내용 : 의사 전달자가 원하는 반응을 얻어낼 수 있는 메시지의 소구 방향이나 주제

- 소구 방법
  - 이성적 소구 : 제품의 구매가 충중이 원하는 편익을 제공함을 강조
  - 감성적 소구 : 구매를 유도할 수 있는 부정적 또는 긍정적 감성을 유발하려는 노력
  - 도덕정 소구 : 옳고 그른 행동을 생각하게 함

2 메시지 구조

- 메시지의 결론을 제시할 것인가, 아니면 청중에게 맡길 것인가
- 제품의 장점만 말하는 일면적 주장, 아니면 장단점을 같이 말하는 양면적 주장

3 인적 의사전달 경로

- 두 명 이상의 사람들이 직접적으로 서로 의사소통하는 방법
- 의사전달자에 의한 직접 통제가 가능하고 제품의 성격이 고가이며 많은 위험이 지각된 제품의 경우 효과적

메시지 원천의 선택

메시지는 일반적으로 신뢰성이 높은 원천에 의해 전달될 때 설득 효과가 높게 나타남

신뢰성이 높은 메시지원천의 특성

- 전문성
- 진실성
- 매력도

1 광고 목표 설정과 표적 청중의 결정

- 정보전달적 광고
- 설득적 광고
- 비교 광고

3 광고 메시지 결정

- 효과적인 메시지의 조건
- 광고 문안 결정

판매촉진 : 세일즈 프로모션 : 단기적 또느 ㄴ조기의 다량 구매를 유도하기 위해 설계된 다양한 도구

판촉의 장점

- 제조업자에게 단기간에 공급과 수요를 조절할 수 있게 함
- 단기간에 다량

- 소비자 판촉 기법
- 중간상 판촉 기법

비가격 판매 촉진

인적 판매 : 판매원이 목표 고객과 직접 대면하여 자사 제품이나 서비스를 구매하도록 설득하는 커뮤니케이션 활동

B2B에서 중요한 역할을 하는 경우가 많다.

- 각각의 고객에게 최적으로 개인화도니 서비스를 제공할 수 있음
- 쌍방향 커뮤니케이션으로 융통성 있는 상호작용을 할 수 있음

## 14장. 유통 경로 관리

- 유통 : 특정 제품이나 서비스가 소비 또는 사용되도록 하는 과정과 관련된 상호의존적인 조직 호은 개인들의 집합
- 유통경로관리 : 제품이나 서비스가 제조업자에게서 표적시장의 최종 소비자에게 적절한 시간에 적절한 수량이 적절한 장소에 제공되도록 관리하는 것

교보문고는 코너 배치를 어떻게 했는지와 같은 것이 유통이다.

- 총거래수 최소의 원리 : 제조업자와 최종 소비자 사이의 총거래수를 현저히 줄여 구매과정에서의 효율성 증가
- 분업의 원리 : 제조업자를 대신하여 수급 조절 기능, 보관 기능, 배달 기능, 정보 수집 기능 등을 담당하여 경제성을 높임
- 핵심 역량 집중 원리 : 제조업자의 제한된 자원을 핵심역량에 집중투자하기 위해 유통 업무는 유통 경로에 맡기는 전략적인 결정

|유통경로의 효용|내용|
|---|---|
|시간 효용|최종 소비자가 원하는 시간에 제품이나 소비스를 구매하거나 이용할 수 있도록 함|
|장소 효용|최종 소비자가 구매하기 쉬운 장소에 제품이나 서비스를 전달|
|소유 효용|최종 소비자가 생산자나 중간상에게서 제품이나 소비스의 소유권을 이전받는 것|

|2수준 경로||의약품이나 식품|
