## 5. 가지고 있는 컨테너 확인하기

켜져 있는 container 확인

```bash
docker ps
```

꺼져 있는 container까지 확인

```bash
docker ps -a
```

## 6. 삭제

켜져 있으면 끄고 삭제 해야 한다.

## Docker가 설치되어 있는지, 실행 중인지 확인하는 명령어

```bash
sudo systemctl status docker
```
