## F1tenth Docker 사용법

전제

1. local에 f1tenth_gym_ros repository가 clone되어 있다.
2. local에 f1tenth_gym_ros의 docker image인 sim-1와 novnc-1 있다.

방법

1. f1tenth_gym_ros가 clone되어 있는 경로로 이동한다. 저 repository에서 docker image를 제공하기 때문에 이 경로에서 시작할 수 있다.
2. Docker image sim-1와 novnc-1를 실행한다.

```bash
docker compose up
```

또는 Docker Desktop에서 sim-1와 novnc-1의 실행 버튼을 누른다. Docker Desktop을 이용하면 f1tenth_gym_ros 경로로 이동할 필요가 없어서 더 편리하다.

3. sim-1 docker image에 접속한다.

```bash
docker exec -it f1tenth_gym_ros-sim-1 /bin/bash
```

4. Launch file을 실행한다.

```bash
source /opt/ros/foxy/setup.bash
source install/local_setup.bash
ros2 launch f1tenth_gym_ros gym_bridge_launch.py
```

5. Vscode로 sim-1 image에 접속하여 코드를 편집한다.
